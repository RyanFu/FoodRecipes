# FoodRecipes
美食菜谱APP是一款提供美食制作方法大全的应用，其内容基本包含了所有菜品、所有菜系、以及各类人群都事宜的菜谱，总计菜单总数超过4万种类。


Tips:
1.需要在lib中的WinSocialShareTool文件夹中导入shareSDK（支持新浪微博、腾讯QQ和微信功能）;
2.需要在lib中导入libraries，框架为MobAPI相关库
